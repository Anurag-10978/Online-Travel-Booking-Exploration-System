🌍 Tour & Travel Website

A responsive Tour & Travel Website built using HTML, CSS, and JavaScript. This project showcases travel packages, adventure activities, and a contact form for users to explore and connect.

📌 Features
🏠 Home section with attractive hero banner
🎯 Adventure activities section (Bungee Jump, Zip Line, Canoeing)
📦 Travel packages (Manali, Goa, Delhi, Jaipur, Kerala, Darjeeling)
🔍 Search bar functionality
📱 Responsive navigation menu
📩 Contact form for user queries
🎨 Clean and modern UI design
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
Font Awesome Icons
